## 六、协议翻译层

```
┌─────────────────────────────────────────────────────────────────┐
│                    Anthropic → OpenAI 翻译                      │
│                                                                 │
│  Anthropic Request               OpenAI Request                 │
│  ┌──────────────────┐           ┌──────────────────┐            │
│  │ system: "..."     │──→        │ messages[0]      │            │
│  │                   │           │ role: system     │            │
│  │ messages: [       │           │ content: "..."   │            │
│  │   {role, content}]│──→        │                  │            │
│  │                   │           │ messages[1..N]   │            │
│  │ tools: [{         │           │ role: user/assist│            │
│  │   name,           │──→        │                  │            │
│  │   input_schema}]  │           │ tools: [{        │            │
│  │                   │           │   type: function │            │
│  │ tool_choice: {    │           │   function: {    │            │
│  │   type: auto/any/ │──→        │     name,       │            │
│  │   tool/none }     │           │     parameters } │            │
│  └──────────────────┘           │   }]             │            │
│                                  │                  │            │
│  Anthropic Response              │ tool_choice:     │            │
│  ┌──────────────────┐           │   auto/required/ │            │
│  │ event: msg_start │←──        │   {type:func}/   │            │
│  │ event: content_  │           │   none           │            │
│  │   block_start    │           └──────────────────┘            │
│  │ event: content_  │                                           │
│  │   block_delta    │           OpenAI Response                 │
│  │ event: content_  │           ┌──────────────────┐            │
│  │   block_stop     │           │ SSE chunk stream │            │
│  │ event: msg_delta │←──        │ data: {delta:    │            │
│  │ event: msg_stop  │           │   {content/      │            │
│  └──────────────────┘           │    tool_calls}}  │            │
│                                  │ data: [DONE]     │            │
│  ⚠️ tool_choice 枚举穷举:        └──────────────────┘            │
│     auto → "auto"                                               │
│     any  → "required"                                           │
│     tool → {"type":"function","function":{"name":"..."}}        │
│     none → "none"  ← 之前漏了，已修复                            │
└─────────────────────────────────────────────────────────────────┘
